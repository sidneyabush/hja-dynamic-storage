# Legacy Code

This folder keeps older scripts from earlier phases of the project

Some of them helped shape parts of the analysis or were used for comparison while methods were being worked out

They are kept here for reference and backtracking during revision work

`run_all.R` does not call anything in this folder, and the main code does not need these scripts to recreate the paper results

The current code lives in the top-level folders:

- `00_data_preprocessing/`
- `01_storage_calcs/`
- `02_analysis/`
- `03_plots/`
- `helpers/`
